# How to Create a Beautiful PDF Deck from the HTML Presentation

## Option 1: Quick PDF Export (5 minutes) - RECOMMENDED

### Method A: Chrome/Edge Print to PDF

1. **Open the presentation**
   - Double-click `El_Dorado_Presentation.html`
   
2. **Open Print Dialog**
   - Press `Ctrl + P` (or `Cmd + P` on Mac)
   
3. **Configure Print Settings**
   - **Destination**: Select "Save as PDF" or "Microsoft Print to PDF"
   - **Layout**: Portrait
   - **Pages**: All
   - **Margins**: None (or Minimum)
   - **Background graphics**: ‚úÖ Checked (IMPORTANT - keeps colors!)
   - **Scale**: 100%

4. **Save**
   - Click "Save"
   - Name it: `El_Dorado_Partnership_Proposal.pdf`
   - Location: Same folder

**Result**: Beautiful 16-page PDF with all colors, formatting, and design intact!

### Method B: Firefox Print to PDF

1. Open `El_Dorado_Presentation.html` in Firefox
2. Press `Ctrl + P`
3. Select "Save to PDF"
4. Check "Print backgrounds" (IMPORTANT!)
5. Save as `El_Dorado_Partnership_Proposal.pdf`

## Option 2: Convert to PowerPoint (.pptx)

### Using Online Tool (Free)

1. **First create PDF** (use Option 1 above)
2. **Go to**: https://www.adobe.com/acrobat/online/pdf-to-ppt.html
   - OR: https://www.ilovepdf.com/pdf_to_powerpoint
3. **Upload** your PDF
4. **Download** the .pptx file
5. **Open in PowerPoint** and edit as needed

**Result**: Editable PowerPoint deck you can customize!
