# SOP: Using Brave Search (Research)

**Goal**: Gather external context and validate assumptions.

## When to Search
1. **New Domain**: "What is the best CRM for real estate?"
2. **Technical Validation**: "How to fix Replit 403 error python"
3. **Competitive Analysis**: "Competitors of [Client Name]"

## Research Loop
1. **Broad Search**: Get overview (3 results)
2. **Deep Dive**: Follow specific URLs if needed
3. **Synthesize**: Combine into internal context

## Constraints
- **Cost**: ~$0.003 per query. Don't spam.
- **Rate Limit**: ~1 request/sec default.
